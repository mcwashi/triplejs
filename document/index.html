ğŸ¨ 2. Editing Theme Colors

All brand colors are set as CSS variables inside assets/css/style.css:

:root {
    --teal: #00c6b1;
    --navy: #0A1533;
    --light: #f8f9fa;
    --gray-text: #6c757d;
}

You can change --teal to replace primary theme color globally.
